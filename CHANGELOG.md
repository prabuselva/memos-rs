## [0.1.0] - 2026-03-15

### Added

- Initial release of memos-rs
- Joplin-like note-taking application
- Markdown support with live HTML preview
- User authentication with session management
- Multi-user support with role-based access
- Import/export from Tomboy and Gnote formats
- Search functionality for notes
- Note organization with tags, favorites, and archive
- Responsive UI with dark mode support
